Accessibility Statement for Staff-Student Liaison Committee (SSLCom)

This is an accessibility statement from Moaz Elleithy.

Conformance status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. Staff-Student Liaison Committee (SSLCom) is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Technical specifications

Accessibility of Staff-Student Liaison Committee (SSLCom) relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • CSS

These technologies are relied upon for conformance with the accessibility standards used.

Limitations and alternatives

Despite our best efforts to ensure accessibility of Staff-Student Liaison Committee (SSLCom), there may be some limitations. Below is a description of known limitations, and potential solutions. Please contact us if you observe an issue not listed below.

Known limitations for Staff-Student Liaison Committee (SSLCom):

  1. It is not responsive with devices with a width of less than 346px because we do not expect our users to use the website with a really narrow phone. The content of the pages contain huge amounts of information that need a bigger screen.
